# To-Do Backend API

A REST API built with Node.js and Express for managing tasks.

## Features
- Get all tasks
- Create a task
- Update a task
- Delete a task

## Tech Stack
- Node.js
- Express.js

## API Endpoints

GET /tasks -> Get all tasks  
POST /tasks -> Create a task  
PUT /tasks/:id -> Update task  
DELETE /tasks/:id -> Delete task
